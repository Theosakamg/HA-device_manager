description: "Devices assigned to this room, with quick access to their full detail."
---

This section lists all devices physically located in this room. Each row shows the device status, MAC address, function and position name.

## Actions

- **Click a row** to open the Devices view pre-filtered on that device's MAC address
- **Enabled / Disabled** status is indicated by the coloured dot on the left

## Columns

| Column | Description |
|---|---|
| ● | Enabled (green) or disabled (grey) |
| MAC | Hardware MAC address — the unique device identifier |
| Function | Functional role (e.g. `light`, `shutter`, `sensor`) |
| Position | Human-readable position within the room |

> **Tip:** To add or move devices between rooms, use the **Devices** view and edit the device's room assignment directly.
