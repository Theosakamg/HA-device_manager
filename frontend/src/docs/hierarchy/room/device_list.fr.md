description: "Équipements assignés à cette pièce, avec accès rapide à leur détail complet."
---

Cette section liste tous les équipements physiquement situés dans cette pièce. Chaque ligne affiche le statut, l'adresse MAC, la fonction et le nom de position de l'équipement.

## Actions

- **Cliquer sur une ligne** ouvre la vue Équipements pré-filtrée sur l'adresse MAC de cet équipement
- Le statut **Activé / Désactivé** est indiqué par le point coloré à gauche

## Colonnes

| Colonne | Description |
|---|---|
| ● | Activé (vert) ou désactivé (gris) |
| MAC | Adresse MAC matérielle — identifiant unique de l'équipement |
| Fonction | Rôle fonctionnel (ex : `light`, `shutter`, `sensor`) |
| Position | Libellé de position dans la pièce |

> **Conseil :** Pour ajouter ou déplacer des équipements entre pièces, utilisez la vue **Équipements** et modifiez directement l'assignation de pièce de l'équipement.
